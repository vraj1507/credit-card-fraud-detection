# Data Directory

This folder should contain `creditcard.csv` from the Kaggle Credit Card Fraud Detection dataset.

**Download:** https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

```bash
# Using Kaggle CLI:
kaggle datasets download -d mlg-ulb/creditcardfraud -p . --unzip
```

The CSV file is not tracked in git due to size (~144 MB) and licensing.
